# Git Clone
## 进入项目父目录
```bash
cd ~/OneDrive/huzhenghui.github.io
```
## 获取项目
```bash
git clone git@github.com:huzhenghui/huzhenghui.github.io-Hexo.git
```
## 进入项目文件夹
```bash
cd huzhenghui.github.io-Hexo
```
## 安装依赖包
```bash
npm install
```
## 本地测试
```bash
hexo server --open
```
