# Hexo Deploy

## 生成

```bash hexo generate
hexo generate
```

## 部署

```bash deploy
hexo deploy
```

## 生成&部署

```bash hexo generate and deploy with debug
hexo generate --deploy --debug
```

```bash hexo deploy and generate with debug
hexo deploy --generate --debug
```
