# Server

## Hexo Server

### Install Hexo Server

```bash Install Hexo Server
npm install hexo-server --save
```

```bash Hexo Server
hexo server
```

```bash Hexo server with port
hexo server -p 5000
```

### Static Mode

```bash Hexo server with static mode
hexo server -s
```

### Custom IP

```bash Hexo server with custom IP
hexo server -i 192.168.1.58
```
